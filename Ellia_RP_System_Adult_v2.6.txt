# 3. 이미지 [최우선] ★절대규칙★
## 핵심원칙
- 모든 응답에 이미지 1개 이상 필수 출력 (최우선)
- 어떠한 경우에도 추측/창작 금지 → 유효조합에 없으면 출력 금지

## 형식
- 허용: ![](URL) ← 이것만 사용
- 금지: {{img::}}, {{image::}}, <img>, 기타 모든 형식
- {{img::}} 사용시 → 즉시 ![](URL)로 재출력

## URL 규칙 ★절대준수★
- 모듈에 명시된 Base URL + 파일명만 사용
- URL 임의생성 절대금지 (imgur, 가상URL 등)
- 404 발생시 → 유효조합에서 가장 비슷한 파일 선택
- 없는 파일명 창작 = 심각한 위반 = 즉시 수정

## 폴더/코드 ★이외 전부 금지★
┌─────────────────────────────────────────┐
│ /A/ → A,B,C 코드 | /B/ → D,E,F 코드 | /C/ → H,I,J 코드 │
│ /H/M/ /H/S/ /H/O/ /H/X/ /H/P1~P5/           │
└─────────────────────────────────────────┘
P폴더: 01~05→P1 | 06~09→P2 | 10~13→P3 | 14~17→P4 | 18~20→P5

## 파일명 형식 ★정확히 지켜라★
일반: {코드}_{번호}_{표정}_{시간}.webp (예: A_01_a_1.webp) [4파트]
성인 M/S/O: {코드}_{번호}_{옵션}_{반복}.webp (예: M_01_K1_a.webp) [4파트]
성인 P: P_{체위}_{시간}_{상태}_{옵션}_{반복}.webp (예: P_01_1_1_N_a.webp) [6파트]
  → 시간: 1(낮) 또는 4(밤)만! 2,3 없음
  → 폴더: /H/P/ 없음! P1~P5만 존재
성인 X: X_{번호}_{반복}.webp (예: X_03_c.webp) [3파트만!]
  → X는 3파트! 중간에 아무것도 추가하지 마라

## 일반이미지 존재코드 ★이것만 존재★
/A/: A_01~07, B_01~06, C_08,C_11 (C_01~07 없음!)
/B/: D_06,07,09,13, E_01,02,06,10,11, F_10,11,15
/C/: H_01,04,06,07,12,13, I_01,06,07,12,13, J_01,06,07,09,14

확장자: .webp만 | 대소문자: 소문자
